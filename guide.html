Obtenir sa clé OpenRouter

La clé donne accès aux modèles d'IA gratuits. Elle est à toi, reste dans ton navigateur, et ne coûte rien si tu utilises uniquement les modèles gratuits.

Suis les 5 étapes ci-dessous.

Créer un compte OpenRouter

Rends-toi sur OpenRouter et inscris-toi avec la méthode de ton choix (Google, GitHub, ou une adresse e-mail). C'est gratuit et rapide.

Ouvrir openrouter.ai

Accéder à ton espace de travail

En haut à droite, clique sur ton profil (Personal) puis sur Workspaces. Si c'est ta première visite, suis l'assistant pour créer ton premier espace de travail.

Personal
📦 Workspaces

Ouvrir le menu « API Keys »

Dans le menu de gauche de ton espace de travail, clique sur API Keys. C'est là qu'on gère les clés d'accès.

🔑 API Keys
🛡️ Guardrails
🗄️ BYOK

Créer une nouvelle clé

Clique sur + New Key, puis remplis la fenêtre :

  • Name : un nom au choix, par exemple « Prysmial Chat ».
  • Credit limit : mets 0 (ou laisse vide). Voir l'astuce plus bas.
  • Expiration : laisse No expiration.
Créer une clé + New Key
Name  ·  ex. « Prysmial Chat »
Credit limit  ·  0
Astuce · limite à 0

Les modèles gratuits (suffixe :free) ne consomment aucun crédit. Une limite à 0 ne les bloque donc pas, et garantit que ta clé ne pourra jamais dépenser d'argent par accident. C'est le réglage le plus sûr.

Clique enfin sur Create.

Copier ta clé

OpenRouter affiche ta clé une seule fois. Copie-la tout de suite (bouton de copie), puis colle-la dans Prysmial Chat.
