## C'est quoi ?

Prysmial Chat est une page web pour discuter avec des intelligences artificielles
parmi les meilleures du moment, gratuitement. Vous l'ouvrez, vous choisissez à quoi
elle doit servir, et vous posez vos questions. Elle sait coder, rédiger, traduire,
expliquer, résumer ou simplement discuter.

Pas de compte à créer chez nous, pas d'abonnement, rien à télécharger. Tout se passe
dans votre navigateur.

## Comment l'utiliser

1. Ouvrez **[chat.prysmial.com](https://chat.prysmial.com)**.
2. Collez votre clé d'accès gratuite (voir juste en dessous comment l'obtenir).
3. Choisissez un type d'usage et un modèle, puis écrivez.

Pour un premier essai, choisissez « Chatbot général ».

## Obtenir votre clé d'accès (gratuit, 2 minutes)

Les IA fonctionnent grâce à un service appelé OpenRouter, qui les regroupe toutes au
même endroit. Pour les utiliser, il vous faut une clé gratuite, un peu comme un mot
de passe d'accès. Elle est à vous et reste sur votre appareil.

Un guide illustré est intégré : cliquez sur **« Comment obtenir une clé gratuite ? »**
sur la page d'accueil. En bref :

1. Créez un compte sur [openrouter.ai](https://openrouter.ai).
2. Ouvrez la section **API Keys**.
3. Cliquez sur **New Key**, donnez-lui un nom, laissez la limite de crédit à **0**.
4. Copiez la clé et collez-la dans Prysmial Chat.

**Est-ce vraiment gratuit ?** Oui. Les modèles proposés ne coûtent rien, et mettre la
limite à 0 garantit que rien ne pourra jamais vous être facturé.

Bon à savoir : ces modèles gratuits sont très demandés. S'ils sont occupés (un message
vous le dira), réessayez un peu plus tard ou changez de modèle.

## Vos données restent chez vous

Nous ne stockons rien.

- Votre clé reste dans votre navigateur. Nous ne la voyons jamais.
- Vos messages vont directement de votre navigateur vers l'IA, sans passer par nos serveurs.
- Votre historique, si vous l'activez, est enregistré sur votre appareil et nulle part ailleurs.

Le détail est expliqué dans l'application, via le lien **« Comment sont gérées mes données ? »**.

## Pour aller plus loin

### Lancer le chatbot avec un outil de codage

Si vous utilisez un assistant de codage (Claude Code, Cursor, etc.), donnez-lui ce
message pour qu'il installe et ouvre le chatbot chez vous :

```text
Télécharge ces deux fichiers dans un même dossier :
- https://raw.githubusercontent.com/Prysmialmain/prysmial-free-chatbot/main/index.html
- https://raw.githubusercontent.com/Prysmialmain/prysmial-free-chatbot/main/guide.html
Puis ouvre index.html dans mon navigateur. C'est une page autonome, sans installation
ni dépendance. Ne modifie rien : je collerai ensuite ma clé OpenRouter dans l'interface.
```

### Choisir vos propres modèles

La liste des modèles se trouve en haut du fichier `index.html`, dans `CATEGORIES`.
Modifiez-la pour ajouter ou retirer des modèles. La liste à jour des modèles gratuits
est sur [openrouter.ai/models](https://openrouter.ai/models?max_price=0).

### Utiliser le fichier directement

Téléchargez `index.html` et `guide.html`, placez-les dans le même dossier, et ouvrez
`index.html` dans votre navigateur. Aucune installation. Une connexion internet reste
nécessaire pour parler à l'IA.
